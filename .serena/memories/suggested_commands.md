Core commands:
- `cp .env.example .env`
- `mkdir -p database`
- `npm install`
- `npm run dev` to run server + client
- `npm run dev:server` for Express API on 3000
- `npm run dev:client` for Vite on 5173 after API health is up
- `npm run test:run` for one-shot unit/integration tests
- `npm test` for Vitest watch mode
- `npm run test:e2e` for Playwright
- `npm run lint` for ESLint
- `npm run pretty` for Prettier check
- `npm run build` for full production build
- `npm run pretty && npm run lint && npm run test:run && npm run build` for full verification
Useful shell tools in this Linux workspace: `git`, `ls`, `cd`, `rg`, `find`, `sed`, `cat`, `npm`.
