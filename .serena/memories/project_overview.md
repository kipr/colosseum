Colosseum is a tournament management and scoring platform for competition events. Stack: React 19 frontend, Express 5 backend, TypeScript throughout, Vite for client dev/build, SQLite via better-sqlite3 for dev/test, PostgreSQL intended for production session/storage use. Core domains: events, teams, seeding rounds, seeding rankings, double-elimination brackets, queue management, customizable scoresheet templates, score review/acceptance, spectator/public views, Google OAuth admin login. Rough structure: `src/client` for React pages/components/contexts, `src/server` for Express routes/services/database, `tests` and `e2e` for verification, `tools` for utilities. Judges submit scores through access-code based scoresheets; admins manage events, queue, brackets, and acceptance workflows.
